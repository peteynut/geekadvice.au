I'm **Peteyyy** — a software engineer, hobbyist game developer, and occasional music producer
based in Australia. This is my personal site.

## What you'll find here

- **Race Wars** — the racing game I'm building in Unreal Engine.
- **Mac gaming** — what works, what doesn't, and what tools help.
- **Music** — the stuff I make, mostly on SoundCloud.
- **Code** — things I learned the hard way, written down so you don't have to.

## How the site works

This site is hand-built with **Next.js** and **MDX**, version-controlled in **Git**, and hosted
on **Azure Static Web Apps** as a static export. There is no database, no admin panel, no
comment form. Every post is a `.mdx` file in the repository. Pushing to `main` deploys in about
a minute.

That choice is deliberate. The less surface area the site has, the less can go wrong.
