A few hours into the journey and it's coming together nicely. There is still plenty to figure out, a basic body shell is certainly the easiest part of this endeavour.

## Blueprint woes
It would appear I didn't spend long enough trying to find and setup my blueprint which I am sure is a common rookie mistake. It looked good when I initially lined it up. But a few hours into the build and I've noticed
some issues lining up certain parts, like the top front/back view is a little offset down, the top view is a little different sized to the side etc. But being that I am not trying to make a 1:1 replica this is fine. I can
fudge the shape in bits that don't line up anyway.
