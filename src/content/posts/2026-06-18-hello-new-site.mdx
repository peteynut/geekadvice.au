If you're reading this, you're on the new site. I rebuilt it from scratch, dropped WordPress,
and switched to a static site hosted on Azure.

## Why

- WordPress is a constant security concern.
- I wanted posts to live in Git so I can write from anywhere with a text editor.
- A static site is less likely to cause me headaches.

## What you'll find

Same content, better layout, faster pages. There's a new **MacOS Gaming** section, an RSS feed,
and tag pages. If something is broken, I'll get around to it maybe.
